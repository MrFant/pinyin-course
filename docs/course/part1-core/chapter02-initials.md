# 第2章：声母系统

## 学习目标
- 掌握21个声母的正确发音方法
- 了解声母的分类体系（发音部位、发音方法）
- 区分易混淆声母对
- 能够准确拼读并运用声母

## 2.1 声母概述

### 什么是声母

声母是汉语音节开头的辅音。每个汉语音节都由声母和韵母两部分组成。声母决定发音的起点位置，韵母决定发音的延续部分。

### 声母表（21个声母）

| 类别 | 声母 | 发音部位描述 |
|------|------|--------------|
| 双唇音 | b, p, m | 上下唇配合发音 |
| 唇齿音 | f | 上齿与下唇配合发音 |
| 舌尖前音 | z, c, s | 舌尖抵住或接近上齿背 |
| 舌尖中音 | d, t, n, l | 舌尖抵住上齿龈 |
| 舌尖后音 | zh, ch, sh, r | 舌尖翘起抵住或接近硬腭前部 |
| 舌面音 | j, q, x | 舌面前部抵住或接近硬腭前部 |
| 舌根音 | g, k, h | 舌根抵住或接近软腭 |

### 声母分类体系

**按发音部位分类：** 发音时，哪一部位的器官形成阻碍，就叫什么音。

**按发音方法分类：**
- **塞音**：发音器官完全阻塞气流，然后突然放开（b, p, d, t, g, k）
- **擦音**：发音器官留有窄缝，气流从缝隙中摩擦而出（f, h, x, sh, s, r）
- **塞擦音**：先完全阻塞，再留窄缝让气流摩擦而出（j, q, zh, ch, z, c）
- **鼻音**：口腔阻塞，气流从鼻腔通过（m, n）
- **边音**：舌尖抵住上齿龈，气流从舌头两边通过（l）

**按送气与否分类：**
- **送气音**：发音时有较强气流喷出（p, t, k, q, ch, c）
- **不送气音**：发音时气流较弱（b, d, g, j, zh, z）

**按清浊分类：**
- **清音**：声带不振动（b, p, f, d, t, g, k, h, j, q, x, zh, ch, sh, z, c, s）
- **浊音**：声带振动（m, n, l, r）

### b - 双唇不送气清塞音

**发音方法：**
1. 双唇紧闭，完全阻住气流
2. 突然打开双唇，让气流冲出
3. 声带不振动，气流较弱（不送气）

**口型图示：**
```
   闭唇阶段    →    打开阶段
   ═══════         ▽
  (双唇紧闭)    (突然张开)
```

**舌头位置：**
```
   ┌──────────────┐
   │  上腭 (硬)    │
   │              │
   │  舌面平放    │  ← 舌头自然平放，不参与发音
   │              │
   └──────────────┘
```

**气流测试：** 将手掌放在嘴前约2厘米处，发"b"音，手掌感觉不到明显气流冲击。

**常见错误：**
- ❌ 把b发成p：送气太强，手掌能感觉到明显气流
- ❌ 嘴唇没有完全闭合就发音：气流从缝隙漏出
- ✅ 正确：双唇完全闭合，突然打开，气流弱

**练习词语：**
- 八 (bā) - 数字8
- 爸 (bà) - 爸爸
- 拔 (bá) - 拔河
- 把 (bǎ) - 把手
- 不 (bù) - 不是
- 笔 (bǐ) - 笔

**记忆口诀：** "八百标兵奔北坡"——全部以b开头的字连起来练习。

### m - 双唇浊鼻音

**发音方法：**
1. 双唇紧闭，完全阻住口腔气流
2. 软腭下降，打开鼻腔通道
3. 气流从鼻腔通过，声带振动

**口型图示：**
```
   ┌──────────────────┐
   │    鼻腔通道       │ ← 气流从这里通过
   │  ════════════    │
   │    口腔阻塞       │ ← 双唇紧闭
   │  ════════════    │
   │    (双唇闭合)     │
   └──────────────────┘
```

**舌头位置：**
```
   ┌──────────────┐
   │  上腭        │
   │              │
   │  舌面平放    │  ← 舌头自然平放
   │              │
   └──────────────┘
```

**关键特征：** m是鼻音，发音时用手捏住鼻子会感觉发不出声音，这是检验鼻音的好方法。

**常见错误：**
- ❌ 把m发成b：气流没有走鼻腔
- ❌ 双唇没有完全闭合
- ✅ 正确：双唇闭合，气流从鼻腔通过，有明显的鼻腔共鸣

**练习词语：**
- 妈 (mā) - 妈妈
- 马 (mǎ) - 马
- 买 (mǎi) - 买东西

**记忆口诀：** "妈妈买马"——mā mā mǎi mǎ，三个m开头的字。

### f - 唇齿清擦音

**发音方法：**
1. 上齿轻轻咬住下唇内缘
2. 气流从上齿和下唇之间的窄缝中摩擦而出
3. 声带不振动

**口型图示：**
```
   ┌────────────────┐
   │  上齿          │ ← 轻触下唇
   │  ╲             │
   │   ╲←── 窄缝 ──│← 气流从这里摩擦而出
   │    ╲           │
   │  下唇          │
   └────────────────┘
```

**舌头位置：**
```
   ┌──────────────┐
   │  上腭        │
   │              │
   │  舌面平放    │  ← 舌头不参与发音
   │              │
   └──────────────┘
```

**气流特征：** 与b、p不同，f是持续发音的擦音，气流从窄缝中持续摩擦而出，不是突然释放。

**常见错误：**
- ❌ 上齿没有接触下唇：变成了h
- ❌ 上齿咬下唇太紧：气流受阻
- ✅ 正确：上齿轻触下唇内缘，气流从缝隙中持续摩擦

**练习词语：**
- 发 (fā) - 发展
- 法 (fǎ) - 法律
- 飞 (fēi) - 飞机
- 风 (fēng) - 风
- 夫 (fū) - 夫人
- 佛 (fó) - 佛像

**记忆口诀：** "发飞机"——fā fēi fēi，感受齿唇的摩擦。

### d - 舌尖中不送气清塞音

**口型图示：**
```

   │  上齿龈(牙床)  │ ← 舌尖抵住这里

   │  舌尖          │

   │  ║             │
   │  舌尖          │ ← 舌尖上翘抵住上齿龈
   │  ╲             │
   │   舌面         │
   └────────────────┘
```

**关键区别：** 与b的发音位置不同——b是双唇音，d是舌尖中音，舌头要主动参与。

**常见错误：**

## 2.3 声母辨析

### 送气与不送气对比

这是声母学习中最基本的对比维度。送气音发音时有较强气流喷出，不送气音气流较弱。

| 不送气音 | 送气音 | 区别要点 | 示例对比 |
|----------|--------|----------|----------|
| b | p | 气流强弱 | 八(bā) - 爬(pá) |
| d | t | 气流强弱 | 大(dà) - 他(tā) |
| g | k | 气流强弱 | 哥(gē) - 科(kē) |
| j | q | 气流强弱 | 鸡(jī) - 七(qī) |
| zh | ch | 气流强弱 | 知(zhī) - 吃(chī) |
| z | c | 气流强弱 | 字(zì) - 次(cì) |

**测试方法：** 将手掌放在嘴前约2厘米处，发不送气音时手掌几乎感觉不到气流，发送气音时手掌能感觉到明显的气流冲击。

### 平舌音与翘舌音对比

这是中国南方学习者最容易混淆的对比维度。

| 平舌音 | 翘舌音 | 区别要点 | 示例对比 |
|--------|--------|----------|----------|
| z | zh | 舌尖位置 | 资(zī) - 知(zhī) |
| c | ch | 舌尖位置 | 此(cǐ) - 吃(chī) |
| s | sh | 舌尖位置 | 思(sī) - 诗(shī) |

**区别要点：**
- **平舌音**（z, c, s）：舌尖平伸，抵住或接近上齿背
- **翘舌音**（zh, ch, sh）：舌尖翘起，抵住或接近硬腭前部

**练习方法：** 先发"资"(zī)，保持口型不变，将舌尖向上翘起，就变成了"知"(zhī)。反复体会舌尖位置的变化。

### 鼻音与边音对比

| 鼻音 | 边音 | 区别要点 | 示例对比 |
|------|------|----------|----------|
| n | l | 气流通道 | 你(nǐ) - 李(lǐ) |

**区别要点：**
- **鼻音n**：口腔阻塞，气流从鼻腔通过。捏住鼻子会感觉发不出声音。
- **边音l**：舌尖抵住上齿龈，气流从舌头两边通过。捏住鼻子仍然能发出声音。

### j, q, x 与 z, c, s 对比

| 舌面音 | 舌尖前音 | 区别要点 | 示例对比 |
|--------|----------|----------|----------|
| j | z | 舌头参与部位 | 鸡(jī) - 资(zī) |
| q | c | 舌头参与部位 | 七(qī) - 此(cǐ) |
| x | s | 舌头参与部位 | 西(xī) - 思(sī) |

**区别要点：**
- **舌面音**（j, q, x）：舌面前部抵住或接近硬腭前部
- **舌尖前音**（z, c, s）：舌尖平伸抵住或接近上齿背

### 常见发音错误总结

| 错误类型 | 错误表现 | 正确方法 |
|----------|----------|----------|
| 平翘舌不分 | z-zh, c-ch, s-sh 混淆 | 注意舌尖位置：平舌平伸，翘舌上翘 |
| 鼻音边音混淆 | n-l 混淆 | n走鼻腔，l走口腔两侧 |
| 送气不送气混淆 | b-p, d-t, g-k 混淆 | 用手掌测试气流强弱 |
| 舌面舌尖混淆 | j-q-x 与 z-c-s 混淆 | j-q-x用舌面，z-c-s用舌尖 |

## 2.5 记忆技巧

### 声母表记忆口诀

> **b** **p** **m** **f**，
> **d** **t** **n** **l**，
> **g** **k** **h**，
> **j** **q** **x**，
> **zh** **ch** **sh** **r**，
> **z** **c** **s**，
> 加上 **y** **w** 共二十三。

（注：y和w是隔音字母，不是真正的声母，但有时会一起教学。）

### 发音部位记忆法

将声母按发音部位分组记忆：

1. **唇音组**（从外到内）：b p m f
2. **舌尖中音组**：d t n l
3. **舌根音组**（最里面）：g k h
4. **舌面音组**：j q x
5. **翘舌音组**：zh ch sh r
6. **平舌音组**：z c s

### 送气/不送气记忆法

不送气音（6个）：b d g j zh z
送气音（6个）：p t k q ch c

规律：不送气音和送气音是成对出现的。

### 易混淆声母记忆技巧

**平翘舌记忆：**
- 平舌音（z c s）：舌头**平**伸，像一条**平**路
- 翘舌音（zh ch sh r）：舌头**翘**起，像一个**翘**起的尾巴

**鼻音边音记忆：**
- 鼻音n：气流走**鼻**腔，捏鼻子发不出
- 边音l：气流走舌头**两**边，捏鼻子也能发

## 常见问题

**Q：为什么我总是分不清平舌音和翘舌音？**
A：平舌音（z c s）和翘舌音（zh ch sh）的区别在于舌尖的位置。平舌音舌尖平伸抵住上齿背，翘舌音舌尖翘起抵住硬腭前部。可以通过反复对比练习来区分。

**Q：送气音和不送气音有什么简单的区分方法？**
A：最简单的方法是用手掌测试。将手掌放在嘴前约2厘米处，发不送气音（如b）时手掌几乎感觉不到气流，发送气音（如p）时手掌能感觉到明显的气流冲击。

**Q：n和l总是分不清怎么办？**
A：n是鼻音，气流从鼻腔通过，捏住鼻子会感觉发不出声音；l是边音，气流从舌头两边通过，捏住鼻子仍然能发出声音。可以先练习捏鼻子和不捏鼻子的区别，然后逐步过渡到正常发音。
