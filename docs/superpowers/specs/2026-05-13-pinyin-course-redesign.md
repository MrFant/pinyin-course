# 汉语拼音打字课程 - 完整重设计文档

## 1. 项目概述

### 1.1 项目目标
创建一套完整、专业的汉语拼音打字课程，包含详细的教学内容、正确的发音系统、丰富的视觉元素和多样化的交互练习。

### 1.2 当前问题
- **课程内容太少**：只有381行，内容简略
- **发音功能错误**：Web Speech API朗读的是英文，不是汉语拼音
- **视觉元素不足**：需要更多动画和交互图示
- **交互体验简单**：练习形式单一

### 1.3 设计目标
- 课程内容：每个章节500-1000行详细内容
- 发音系统：使用pinyin-pronunciation库，支持所有声母、韵母、声调
- 视觉设计：教育专业风格，清晰图表
- 交互练习：闪卡、打字、听音辨音、声调判断4种形式

## 3. 发音系统设计

### 3.1 技术方案
使用 `pinyin-pronunciation` 库，这是一个开源的汉语拼音发音库。

### 3.2 功能特性
- 支持所有21个声母发音
- 支持所有39个韵母发音
- 支持四声声调变化
- 支持音节组合发音
- 离线可用，无需网络

### 3.3 集成方式
```javascript
// 安装
npm install pinyin-pronunciation

// 使用示例
import { speakPinyin } from 'pinyin-pronunciation'

// 朗读声母
speakPinyin('b')

// 朗读韵母
speakPinyin('a')

// 朗读带声调的拼音
speakPinyin('mā')  // 第一声
speakPinyin('má')  // 第二声
speakPinyin('mǎ')  // 第三声
speakPinyin('mà')  // 第四声
```

### 3.4 发音组件设计
创建 `usePronunciation` hook，封装发音功能：
- `pronounceInitial(initial)` - 朗读声母
- `pronounceFinal(final)` - 朗读韵母
- `pronounceSyllable(syllable)` - 朗读音节
- `pronounceTone(character, tone)` - 朗读带声调的字

    pronunciation: '双唇紧闭，突然打开，气流较弱'
  }

### 5.2 打字练习
**功能：**
- 显示汉字或拼音
- 用户输入对应拼音
- 实时检查正确性
- 统计速度和准确率

**实现：**
```javascript
function TypingPractice({ word, onCorrect, onWrong }) {
  const [input, setInput] = useState('')
  const [showAnswer, setShowAnswer] = useState(false)
  
  const checkAnswer = () => {
    if (input.toLowerCase() === word.pinyin.toLowerCase()) {
      onCorrect()
    } else {
      onWrong()
      setShowAnswer(true)
    }
  }
  
  return (
    <div>
      <div className="word">{word.character}</div>
      <input 
        value={input} 
        onChange={(e) => setInput(e.target.value)}
        placeholder="输入拼音..."
      />
      <button onClick={checkAnswer}>检查</button>
      {showAnswer && <div className="answer">{word.pinyin}</div>}
    </div>
  )
}
```
